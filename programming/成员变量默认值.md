





## 成员变量的默认值

|          | 数据类型                      | 默认值   |
| -------- | ----------------------------- | -------- |
| 基本类型 | 整数 (byte, short, int, long) | 0        |
|          | 浮点数 (float, double)        | 0.0      |
|          | 字符 (char)                   | '\u0000' |
|          | 布尔 (boolean)                | false    |
| 引用类型 | 数组、类、接口                | null     |
