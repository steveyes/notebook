

| 不同点         | 成员变量                                               | 局部变量                                               |
| -------------- | ------------------------------------------------------ | ------------------------------------------------------ |
| 在类中的位置   | 类中，方法外                                           | 方法中，方法声明上(形参)                               |
| 作用范围       | 类中                                                   | 方法中                                                 |
| 初始化值       | 有默认值                                               | 没有默认值，必须先定义，复制，再使用                   |
| 再内存中的位置 | 堆内存                                                 | 栈内存                                                 |
| 生命周期       | 随诊对象的创建(入堆)而产生，随着对象的销毁(出堆)而销毁 | 随着方法的调用(入栈)而产生，随着方法的返回(出栈)儿销毁 |
