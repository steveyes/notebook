# 安全上下文

> secure context



| 权限          | 文件                 | 目录                                                  |
| ------------- | -------------------- | ----------------------------------------------------- |
| r(readable)   | 可以查看文件内容     | 可以用`ls`列出目录内部所有文件                        |
| w(writable)   | 可以编辑或删除此文件 | 可以在目录内创建文件、删除、修改等                    |
| x(executable) | 可以执行文件         | 可以用`cd`进入目录，可以使用`ls -l`查看文件内详细信息 |
