# django rest viewset



DRF 请求方法对应表

| action         | 请求方式    | 详细                            |
| -------------- | ----------- | ------------------------------- |
| create         | post        | 创建一条数据                    |
| list           | get         | 获取多条数据                    |
| retrieve       | get /pk/    | 获取单条数据 需带上pk           |
| update         | put /pk/    | 修改单条数据的全部字段 需带上pk |
| partial_update | patch /pk/  | 修改单条数据的某个字段 需带上pk |
| destroy        | delete /pk/ | 删除单条数据 需带上pk           |
